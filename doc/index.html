FastLevelSet ImageJ plugin

Note this plugin is incomplete.

Installation

  1. Copy FastLevelSet_Plugin.jar into a subdirectory of your ImageJ plugins directory.
  2. If ImageJ is already running then restart (or choose Help → Refresh Menus).

Example segmentation

  1. Open example-his1.tif. This is a time-lapse image stack.
  2. Select a circular region of interest (ROI) as shown in the figure.
  3. Open Plugins → [subdirectory] → Fast level set segmentation in the plugins menu
  4. In the Fast level set settings dialog:
    • Keep Use ROI, this means the ROI created in the previous step will be used for the initialisation. The other values values are the built-in ImageJ auto-thresholder methods.
    • Change Iterations to 5. This controls how much the segmentation can evolve from the initialisation.
    • Change Field type to Local region (Hybrid). Region Chan-Vese works well when the foreground and background intensities are fairly constant, however for this example we only want to segment one of the objects.
  5. Click on OK.
  6. Two new windows will appear. One window shows the step-by-step evolution of the level-set, the other shows the final segmentation of each frame.
